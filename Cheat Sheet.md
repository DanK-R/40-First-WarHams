## Actions
### Full Actions:
```dataview
Table Type, Description
From "Actions"
Where contains(Type, "Full Action")
```
### Half Actions:
```dataview
Table Type, Description
From "Actions"
Where contains(Type, "Half Action")
```
### Reactions:
```dataview
Table Type, Description
From "Actions"
Where contains(Type, "Reaction")
```
### Available:
```dataview
Table Type, Description
From "Actions"
Where Available = True
```
